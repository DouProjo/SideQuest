SideQuest — Task‑Tracking Website
🌟 Overview
SideQuest is a lightweight, gamified task‑tracking website designed to make productivity feel fun. Instead of traditional to‑do lists, users complete “quests” — small, achievable tasks that build momentum and keep motivation high. The goal is simple: turn everyday responsibilities into an adventure.

🎮 Features
Create Quests  
Add tasks with titles, descriptions, and optional difficulty levels.

Track Progress  
Mark quests as complete and watch your progress update in real time.

Organize by Category  
Group quests into categories like School, Work, Fitness, or Personal.

XP‑Style Motivation  
Optional experience points system to reward completed quests.

Clean, Responsive UI  
Designed to work smoothly on desktop and mobile.

🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript

Backend: (If applicable) Node.js / Express

Storage: localStorage or database depending on deployment

Hosting: GitHub Pages / Replit / Render
